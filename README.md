A paid-first, Play-Store-safe Android developer utility app.

## Features (v1)
- Logcat Viewer (foreground-only, safe mode)
- Intent Launcher
- App & Package Viewer
- APK Info Viewer (SAF file picker)
- Deep Link Tester
- Clipboard Inspector

## Tech
- Kotlin
- Jetpack Compose + Material 3
- Min SDK: 26
- Target SDK: 34

## Build
```bash
./gradlew assembleDebug
./gradlew assembleRelease

```
